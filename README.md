A Python command-line tool that checks password strength using security rules like length, uppercase, lowercase, numbers, and special characters.

--------

## Features
- Minimum length validation (8+ characters)
- Uppercase, lowercase, number, and special character checks
- Password strength rating from Extremely Weak to Very Strong
- User-friendly feedback

## Tech Used
- Python
- Regular Expressions (re)

## How to Run
```bash
python password_checker.py
